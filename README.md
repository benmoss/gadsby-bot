# gadsby-bot

a simple twitter bot that searches for quality tweets that do not contain the
letter 'e'.

inspired by (http://en.wikipedia.org/wiki/Gadsby_(novel))[http://en.wikipedia.org/wiki/Gadsby_(novel)]
